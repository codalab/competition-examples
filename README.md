Competition Examples
====================

This repository contains competition bundle examples for [Codabench](https://www.codabench.org/).

### Folders

* [codabench/](codabench/): Codabench examples of competitions you can clone to start your own project.

* [codalab/](codalab/): CodaLab Competitions examples (_deprecated_).
