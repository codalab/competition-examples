# MNIST Mini-Bundle: Handwritten Digit Classification


Welcome to the **MNIST Mini-Bundle**! This repository provides a compressed, easy-to-use version of the classic MNIST dataset, specifically designed for quick experimentation and hosting competitive challenges on platforms like **Codabench**.

## 📁 What's in this Folder?

**`Mini-MNIST-Bundle/`**: A ready-to-upload Codabench bundle. It contains the competition configuration, ingestion and scoring programs, the mini dataset organized for evaluation, and utitlites scripts to test bundle locally and compile bundle for codabench.

---

## 📊 The Mini-Dataset

This bundle features a subset of the original MNIST dataset to allow for rapid development and testing (Mini-MNIST).

### Data Statistics:
| Split | Samples | Images per Class | Format |
| :--- | :--- | :--- | :--- |
| **Training Set** | 200 | 20 | PNG Folders + CSV Labels |
| **Test Set** | 50 | 5 | PNG Folders |

**Label Distribution:**
The classes (0-9) are perfectly balanced in this mini-version, ensuring fair evaluation across all digits.

---

## 🚀 Submissions

To make it easier for participants to get started, we have provided two sample submission files:

### 1. Code Submission
- **File**: `mini_MNIST_code_submission.zip`
- **Content**: Contains `model.py` which implements a **Nearest Template Centroid** classifier. This serves as a baseline implementation for participants.

### 2. Result Submission
- **File**: `mini_MNIST_result_submission.zip`
- **Content**: Contains the `predictions.npy` file generated by running the baseline model on the test set. This shows the format required for "result-only" submissions.
