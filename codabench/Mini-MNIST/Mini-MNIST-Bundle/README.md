# Mini-MNIST Competition Bundle

This directory contains all the necessary files and folders to host the **Mini-MNIST Digit Classification Challenge** on Codabench.

## Directory Structure
```text
Competition_Bundle/
├── competition.yaml            # Main configuration file (title, phases, tasks, leaderboard)
├── logo.png                   # Competition logo
├── README.md                  # This file
├── input_data/                # Input data for the tasks
│   ├── train_data/            # 200 training PNG images (20 per class)
│   ├── train_labels.csv       # Filename to Label mapping for training
│   └── test_data/             # 50 test PNG images (5 per class)
├── reference_data/            # Ground truth data for scoring
│   └── test_labels.csv        # Ground truth labels for the test set
├── ingestion_program/         # Code to run participant submissions
│   ├── ingestion.py          # Ingestion logic (loading PNGs, running model)
│   ├── run_ingestion.py      # Entry point for Codabench
│   └── metadata.yaml         # Ingestion metadata
├── scoring_program/           # Code to evaluate predictions
│   ├── score.py              # Scoring logic (accuracy calculation)
│   ├── run_scoring.py        # Entry point for Codabench
│   └── metadata.yaml         # Scoring metadata
├── pages/                     # Documentation pages for the competition
│   ├── overview.md           # General challenge overview
│   ├── evaluation.md         # Explanation of metrics
│   └── terms.md              # Terms and conditions
├── sample_code_submission/    # A sample Python model submission
│   └── model.py              # Baseline model (Nearest Centroid)
├── sample_result_submission/  # Sample predictions generated by the baseline
│   └── predictions.npy       # Numeric predictions for the test set
├── utilities/               # Utility scripts for local testing and bundle creation
│   ├── compile_bundle.py     # Script to package the bundle for upload
│   └── test_bundle_locally.py # Script to test ingestion and scoring locally
```

## How to use this bundle on Codabench

1.  **Compile the zip**: Run `python3 utilities/compile_bundle.py`.
2.  **Upload to Codabench**: Go to your Codabench account and upload the generated zip as a new competition.
3.  **Test**: Ensure the ingestion and scoring programs run successfully on the platform.
