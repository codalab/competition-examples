# Starting kit

Welcome to the starting kit of "Classify Wheat Seeds (result submission)" competition!

You will find here what you need to make your first submission to the competition:

- `dev-submission.zip`: a ready-to-submit file for the development phase
- `final-submission.zip`: a ready-to-submit file for the final phase
- `prediction`: the actual content of an example submission for the development phase. As this is a **result submission** competition, the submission consists in predicted label
- `model.py`: an example machine learning model
- `run.py`: a script to run to train your on the public data, and to make predictions on the test data
