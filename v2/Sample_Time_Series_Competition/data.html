Data and starting kit

Everything you need to get started is in the starting kit, downloadable from the FILES tab.
The starting kit is in Python. It was tested with Anaconda Python 3.6. For an easy start, use: 

 jupyter-notebook README.ipynb
The starting kit includes sample data. You may also download additional "public" data from the FILES tab, for practice purposes.

Data Format

We provide csv files, with temporal samples in lines. The header line indicating the meaning of the columns. The first column is a time index.
